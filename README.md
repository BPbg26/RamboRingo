# RAMBO v RINGO: The Ultimate Battle

A retro browser action game with three levels, bosses, music, and a final ping pong showdown in a German toilet DJ booth.

## Play Now
[Play the Game](https://bpbg26.github.io/RamboRingo/)

## Features
- 3 unique levels
- Boss fights
- Soundtrack
- Final battle ping pong mode
- End credits
- High scores

## Controls
- Move: Arrow Keys / WASD
- Shoot: Space
- Start: Enter / Space
- Return to Menu: R
